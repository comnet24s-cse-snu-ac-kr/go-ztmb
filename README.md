# ZKMBX (w/o zkp)

- Logic implementation without ZKP written in go

## Quickstart

### Demo

```bash
make
./example.sh
```

### Pre-built binaries

- Pre-built binaries are available in [/build](./build) dir
    - `build/zkmbx-darwin-amd64`: x86_64 binary
    - `build/zkmbx-darwin-arm64`: ARM64 binary
